# Dynamic Bullwarks, Fennec Edition
modificações por zats e inu. <p>
Esse mod é uma versão bastante modificada do bullwark, com novos barricadas, balanceamento de loot, nova loja, novos inimigos e novas ondas.

### Nova loja
Loja feita por Smith. Mais itens, divididos em categorias.
### Balanceamento de loot
Agora tem uma diversidade maior para o loot de armas, excluindo itens repetindos, dando chance para uma arma diferente aparecer.
### Supply Drop
- Agora é aleatório qual helicoptero dropara a supply drop. 
- A caixa agora cortará o paraquedas 30 segundos depois de ser solta, diminuindo a chance de cair fora da área da BullwarkCity.
### construção
- adicionado o mobile builder, construa sem precisar estar perto do bulwark
- adicionado rotação (30º por vez) e reset de inclinação 
- veículos agora spawnam mais alto do chão para evitar clipar
- adicionado veiculo para towing, funcional!
### Novas ondas especiais
- Juggernaut, derrote um inimigo quase indestrutivel (75% pronto, WIP)
- helicóptero (ainda WIP)
### Pequenos tweaks
- agora é possível ir para fora da zona do bulwark com veículo, mas se sair dele será levado de volta para a àrea
- HUD agora informa número de inimigos restantes
- Opção de desativar tickets
- Novos sons para início de rodada, civil, etc.
- BLUFOR agora são inimigos comuns de encontrar
